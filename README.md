# Bo Ding's Homepage

Source code for [Bo Ding's personal website](https://beauding.github.io/).

This site is built with Jekyll based on the [AcadHomepage](https://github.com/RayeRen/acad-homepage.github.io) template.

## Local Preview

Install Ruby and Bundler, then run:

```bash
bundle install
bundle exec jekyll serve
```
